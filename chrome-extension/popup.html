• All parameter values should be lowercase.
  • Country code should be a valid two-letter country code.
  • Language should be the name of a language expressed in English. E.g. "french" not "Francais".
  • Language code should be an ISO two-letter language code. E.g. "en" or "fr".
  • Default sale type should match a sale type in the "Filter Sale Type" drop down list. E.g. "DAILY DEAL". Sale type "*" shows all items. Sale type "!*" shows all items on sale. Sale types starting with "!" filter for items on sale that are NOT that sale type.
